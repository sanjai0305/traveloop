# Trip Planner Hackathon

A modern full-stack travel planning web application built using the MERN stack.

Users can:
- Create trips
- Build itineraries
- Manage packing checklists
- Save trip notes
- Organize activities

---

# Features

- JWT Authentication
- Create & Manage Trips
- Dynamic Itinerary Planner
- Packing Checklist
- Trip Notes
- Activities Module
- User Profile
- Responsive Modern UI
- MongoDB Integration

---

# Tech Stack

## Frontend
- React.js
- Tailwind CSS
- React Router DOM
- Lucide React Icons

## Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
